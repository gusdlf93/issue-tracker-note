# Godot
고도 엔진(Godot Engine)은 2D 및 3D 게임 제작을 위한 크로스 플랫폼 게임 엔진이다.

## 지원 플랫폼
 - 데스크탑 : Linux, Mac, Windows
 - 모바일 : Android, iOS
 - 웹 : HTML5

## 라이센스
 - MIT

## 지원언어
 - GDScript(고도스크립트, 자체사용 언어)
 - C++
 - C#
 - Visual Script
